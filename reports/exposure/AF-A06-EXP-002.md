# AF-A06-EXP-002 - Deployed PSR-7 URI-validation exposure

## Classification

- Status: Open exposure; application-specific exploitability unverified
- Triage severity: High
- Exploit severity: Not assigned
- Category: OWASP Web A06 - Vulnerable and Outdated Components
- Target build: `fe8268f8953bc7c9bde9b01020b9ddf8b5c5649d`
- Component: `guzzlehttp/psr7 2.12.1`
- Scanner reference: `GHSA-c2w2-prh8-qm98`

## Description and possible clinical impact

The exact Composer input and running OpenEMR container both reported
`guzzlehttp/psr7 2.12.1`. The pinned scanner associated that version with a weak URI
host-validation advisory. If an OpenEMR request path constructs or trusts a PSR-7 URI
from attacker-controlled host input, the condition could affect destination
validation or downstream authorization assumptions.

No such path was demonstrated in the Clinical Co-Pilot. The primary same-origin
Co-Pilot chat proxy uses a configured sidecar base URL and native PHP cURL. Wider
OpenEMR code imports PSR-7, so installed-component exposure remains, but a package
match is not proof of exploitability.

4. Confirm the running OpenEMR container reports
   `guzzlehttp/psr7 2.12.1`.
5. Confirm the Co-Pilot chat bridge builds its fixed sidecar request with native cURL.
6. Record that no hostile host or URI parser case was sent to the deployed target.

## Expected versus observed

- Expected: no affected deployed URI component remains without reachability and
  remediation disposition.
- Observed: the affected PSR-7 version is installed; no Co-Pilot-specific
  attacker-controlled host path or advisory behavior was established.

## Recommended remediation and validation

1. Upgrade `guzzlehttp/psr7` to a nonaffected, OpenEMR-compatible version.
2. Inventory PSR-7 construction sites and classify which can receive untrusted host,
   authority, or URI text.
3. Add unit/contract tests for ambiguous host forms at any reachable boundary.
4. Rebuild, regenerate the SBOM, rerun the scanner, and verify the container version.
5. Replay the Co-Pilot regression suite after the dependency update.
